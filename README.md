# Idle Ecologist Phaser Starter

This project is a basic Phaser setup using TypeScript. It displays 'Hello, Phaser!' in a simple scene.

## Getting Started

1. Install dependencies:
   npm install

2. Build the project:
   npm run build

3. Start the project:
   npm start

## Project Structure
- src/: TypeScript source files
- public/: Static assets and index.html

## Customization
You can expand the scene in src/main.ts to add your own game logic.

## Requirements
- Node.js
- npm
